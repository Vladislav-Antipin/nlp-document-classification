# NLP Document Classification

This project explores and evaluates different natural language processing (NLP) methods for binary text classification tasks.

## Tasks

Two classification problems are considered:

1. **Movie review sentiment analysis**:  predict whether a given review shows a positive or
negative sentiment.
2. **President speech authorship attribution**: predict whether a sentence belongs to Jacques
Chirac’s or François Mitterrand’s speech.
